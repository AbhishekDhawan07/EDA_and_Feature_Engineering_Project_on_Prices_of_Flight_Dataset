## 📖 Project Overview
This project explores flight pricing data to identify trends, patterns, and key factors affecting ticket prices. It also includes feature engineering techniques to improve data quality and prepare it for machine learning models.

## 📊 Dataset Information
- Dataset Name: **flights_data.csv**
- Contains information about:
  - Airline
  - Source & Destination
  - Duration
  - Number of Stops
  - Price
  - Days left before departure
